# Emotion Detection from Text

Fine-tuned **DistilBERT** model that classifies English text into 6 emotion categories:
sadness, joy, love, anger, fear, surprise.

**Accuracy:** 93.75% on test set | **Dataset:** dair-ai/emotion (20K samples)
